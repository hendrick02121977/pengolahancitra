# Materi Perkuliahan Image Processing dengan Google Colab

Repositori ini berisi materi perkuliahan untuk Image Processing (Pengolahan Citra) yang dirancang untuk pembelajaran interaktif menggunakan Google Colab.

## Daftar Materi

### 1. Dasar-Dasar Image Processing
- Pengenalan Citra Digital
- Representasi Citra
- Operasi Dasar pada Citra

### 2. Filtering dan Konvolusi
- Linear Filtering
- Gaussian Blur
- Edge Detection (Sobel, Canny)

### 3. Segmentasi Citra
- Thresholding
- K-Means Clustering
- Watershed Algorithm

### 4. Transformasi Citra
- Geometric Transformation
- Fourier Transform
- Wavelet Transform

### 5. Fitur Ekstraksi
- Corner Detection
- SIFT, SURF
- HOG (Histogram of Oriented Gradients)

### 6. Analisis Citra Lanjutan
- Morphological Operations
- Color Space Transformation
- Histogram Equalization

## Cara Menggunakan

1. Buka Google Colab: https://colab.research.google.com/
2. Pilih notebook yang ingin dipelajari
3. Klik tombol "Copy to Drive" untuk membuat salinan di Drive Anda
4. Jalankan setiap cell untuk melihat hasil
